## 🤝 How to Find This Collaborative Repository on GitHub

If a collaborator has invited you to this repository, here is how to access it:

### Method 1 — GitHub Dashboard (Recommended)
1. Log in to [github.com](https://github.com).
2. On the left sidebar of your dashboard, under **"Repositories"**, click **"View all"** or scroll down to see recently active repositories — **collaborative repos you have been added to appear here automatically**.

### Method 2 — Your Profile Page
1. Click your avatar (top-right corner) → **"Your profile"**.
2. Click the **"Repositories"** tab.
3. Set the filter to **"All"** (not just "Sources") — repositories you collaborate on will appear in this list.

### Method 3 — Direct URL
Navigate directly to:
```
https://github.com/bobownYAO/COMP1110_Project_2026
```

### Method 4 — Organizations / Shared Access
If the repo was shared through an organization or a team:
1. Click your avatar → **"Your organizations"**.
2. Select the relevant organization.
3. Click **"Repositories"** inside the organization page.

> **Tip:** If you have accepted the collaboration invitation via email, GitHub will redirect you straight to the repository. Check your registered email inbox for the invitation link.
